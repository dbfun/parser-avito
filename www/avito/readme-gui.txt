Требования к окружению
  PHP 5.2+
  MySQL 5.0

По причине долгой работы скрипта GUI-версия (запускается из браузера) не запрограммирована.
Все что написано ниже - на возможное будущее.

Порядок работы

1. Распаковать архив в каталог веб-сервера, например /home/u001/www/

2. Настройка
  2.1. Отредактировать файл avito/config.json (секция mysql) для подключения к базе данных
  2.2. Выполнить скрипт установки, открыв в браузере http://site.lan/avito/install.php
  
3. Парсинг объявлений
  Открыть в браузере http://site.lan/avito/index.php
